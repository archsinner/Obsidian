# Iframes
[[HTML & CSS]]
---

Iframes are HTML elements that display a web page within a web page.

```html
<!DOCTYPE html>
<html>
    <head>
        <title>An Iframe</title>
    </head>
    <body>
        <iframe src="learn-html.org"></iframe>
    </body>
</html>
```

You can also style it using CSS (e.g Removing the border).
